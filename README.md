# Stats File for Non-Native Vowel Classification Study

This is a short R project which takes anonymised participant csv data and uses Multidimensional Scaling (MDS).

## Files Included:

DATA_CONVERSION.Rmd

-   Converts raw CSV data into a matrix

MDS.Rmd\*

-   By Daniel Daidone
-   Source: [ddaidone.com](https://www.ddaidone.com/uploads/1/0/5/2/105292729/mds_analysis.rmd)
-   Transforms participant data into perception maps
-   Shows the amount dimensions/rules used to group sounds
